随笔-2015-06-09

  • 随笔

这是一篇个人随笔.

关于今天

今天下午社团有活动，搞了个优秀毕业生经验交流分享会。我被抓去拍照，兼当苦力。

其实我心里真的一百个不愿意，我觉得社团活动这种东西， 真的不是我们这种技术猿应该干的事情，太繁琐，太没意义。
真的，至少对于我来说，今天下午浪费了的6个小时的有效的连续的时间真的是太宝贵了的。

但是呢？ 我收获了什么呢？ 在这个活动中，我以一种懒散的态度拍了若干张绝对算不上好的照片，还好部长没有叫我写新闻稿，真是够了。 然后其他的所有时间除了干苦力就是在那里无聊的玩手机。真是够浪费的。

我相信，这是我最后一次干社团活动了，以后打死不去，我宁愿在寝室打代码。

再说说那些优秀学长学姐分享的经验吧， 其实对我的用处真的不大， 他们要么是应聘全球四大会计所的经验， 要么是入体制的经验，要么是考研的经验。实习也是干一些乱七八糟的什么营销类市场类的工作，我不否认这些东西其实是有用的，但是对我来说，对于一个技术猿来说， 其实没有太大的意义。

我需要的经验，网络上很多技术猿前辈们都有分享，通过我自己的信息接收渠道，每三天，我就可以阅读一篇这类应聘技巧，工作经验分享类的文章，没必要去听一些跟我兴趣爱好，之类分类相差十万八千里的学长学姐的经验介绍。

其实，在我内心，我一直将自己视为一个搞技术的人，尽管我的技术很差很差，所以我并不是很想走别的同学们走的那一套——例如做班干部，例如混社团。其实过去的我也不是没有想过培养自己那些方面的能力的，但是事实证明，我的确不是那块料。

我知道我的技术能力很差，我要努力的地方也还很多，昨天在空间看到有同学发了这样一条说说，我觉得是非常符合我的，于是我也复制了一下。

时间过得很快，高中毕业一年了，在大学，我没有想过拿奖学金，没有想过当学生干部，只想做自己想做的事，翘想翘的课，学想学的东西，我不喜欢还是拿卷面成绩对比的大学，我喜欢比的是见识，风度，比的是思想的大学，通过大学四年培养起来的思想，能力，气质和内涵才是终生的竞争力，我希望大学毕业后，听到的不是我有多么落魄或者我有多么成功，而是我有多么不一样
转自我亲爱的小伙伴，大家都是有热情的年轻人。

做为一个技术猿，我其实是很不在乎他们那一套的。

关于我自己

昨天，我度过了我20岁的最后一天，是的，我21了。

我从来没有过过一次生日，昨天也是，没有庆祝，这没有必要。

其实我也算是给自己买了生日礼物吧， 我换了新的手机，买了新的鞋子。

其实这也是阴差阳错的，我傻逼的去营业厅换了4G的电信卡，结果我原来的手机由于买的是进口的水货，不是国内的产商生产的，不能接入国内的4G网络。好吧，我不得不换一个新的手机，并且我原来的那个摩托罗拉实在是卡的用不了了。于是我在征求了父亲的同意之后，买了个小米4。顺便提一句，这是我到现在为止，买过的最贵的手机。

至于新的鞋子，此前我只有一双上个学期买的为了过冬的鞋子，现在夏天来了，需要一双新的鞋子很正常，不算什么大事。

然而，我的20年过去了，我现在的状态是怎么样的呢？我这样问自己。

我现在的兴趣在计算机上面，现在在学校参与ACM，偶尔刷刷POJ，再一次学习已经被我遗忘的算法和数据结构。 高中学的那些东西已经完全忘光了。

然后我也在计算机的其他方面有一些业余爱好，例如最新在折腾个人博客。

我初步掌握的技能有：

  • C语言 这是我学的最好的了吧。
  • C++ 好吧，渣渣，虽然现在专业课在上这门课。
  • linux 一些皮毛知识
  • git 基础， 现在主要用github写博客。感觉我还是没学会git的精髓。
  • markdown 基础， 写博客用的， 在实践中得到锻炼。
  • latex公式部分 算入了latex的门了吧，平时写博文的时候有时候要用数学公式来表达。
  • html 入门， 能看懂css和JavaScript。

今天早上新萌发了学习node.js的想法，找机会学学。

我知道我自己是个弱渣，在这个巨大的互联网上，我遇到了很多跟我同龄甚至比我小的人成就比我大的多的，就更不用说以前遇到的听说的那些OI和ACM的算法大神了。

关于课业

真的是心有余而力不足，好吧，我承认有些地方是我自己的原因，例如电路，大物，其实这些学科我是可以学好的，但是不知道为什么，我就是没有用心去学，于是到了现在要挂科的地步，真不知道期末怎么过。。

但是，英语，就真的不是我能控制的，其实就我自己内心而言， 我是很想把英语学好的，毕竟计算机的世界里太多的英文了。但是我感觉我的英语还是要挂，怎么办？

其他的课还好，一些文科方面的东西，例如军概，近纲之流， 背一背就可以过去的， 其实不需要花太多的精力去管。

最后，所有的东西汇成一句话：加油，程序猿！！
